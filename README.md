Описание программы автоматического открытия веб-страниц в Google Chrome
Версия 0.1.3
Описание программы
Программа предназначена для автоматического открытия набора веб-страниц в браузере Google Chrome. Она помогает, когда нужно регулярно запускать несколько сайтов, но не хочется каждый раз вручную вводить адреса. 
Основная также идея создания рабочего места менеджера. Когда он может прийти на работу включить компьютер и у него откроется браузер со всеми ссылками необходимыми ему для работы. 
Основная логика работы
1.	Файлы для хранения данных:
•	urls.txt — список сайтов, которые нужно открыть (каждый адрес в новой строке).
•	opened_urls.txt — журнал уже открытых ссылок с датой и временем их открытия.
2.	Проверка даты при запуске:
•	При старте программа читает opened_urls.txt.
•	Если дата в файле не совпадает с сегодняшней — файл очищается.
•	Таким образом, каждый день программа начинает с «чистого» списка.
3.	Загрузка списка открытых ссылок:
•	После проверки даты загружаются все ранее открытые адреса (если они есть).
•	Это нужно, чтобы не открывать один и тот же сайт несколько раз за день.
4.	Функция is_chrome_running():
•	Проверяет, запущен ли процесс Chrome через библиотеку psutil.
•	Если Chrome не работает — программа запускает его.
•	Если работает — добавляет новые вкладки.
5.	Функция open_url(url):
•	Проверяет, был ли адрес уже открыт сегодня.
•	Если нет — открывает его в новой вкладке Chrome.
